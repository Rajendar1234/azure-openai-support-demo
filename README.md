# Azure OpenAI Support Demo

This is a demo project that shows how AI can be used to assist L1/L2 customer support teams.

## Features
- Simple Python-based support assistant
- Handles common support issues
- Example use case for AI-powered support automation

## Technologies Used
- Python
- Azure OpenAI (conceptual)
- REST API concepts

## Use Case
This project demonstrates how AI can help resolve common customer issues faster and reduce manual effort.

Note: This is a demo project created for learning and portfolio purposes.
